# Student Management System

## Overview

A full-stack Student Management System built using Angular and Node.js. It allows users to manage student records with a simple and responsive interface.

## Tech Stack

- Angular
- Node.js
- Express.js
- Bootstrap

## Features

- Login Page
- Student Dashboard
- View Students
- Add Student
- Delete Student
- Responsive UI

## Project Structure

```
student-management/
├── src/
├── backend/
├── package.json
└── README.md
```

## Installation

### Frontend

```bash
npm install
ng serve
```

Runs on:

```
http://localhost:4200
```

### Backend

```bash
cd backend
npm install
npm run dev
```

Runs on:

```
http://localhost:5000
```

## API Endpoints

- GET /students
- POST /students
- PUT /students/:id
